# Academic Chat Bot (Bilingual)
An interactive, AI-powered academic assistant built using Python. It features robust prompt processing capability to help students with educational resources and queries.

## Features
- **Bilingual Support:** Fully capable of understanding and responding in both **English** and **Urdu** (Roman Urdu/Script) for better accessibility.
- **AI Model Integration:** Leverages generative AI frameworks to provide smart, context-aware academic feedback.
- **Natural Language Parsing:** Smoothly processes student queries and breaks them down into interactive learning modules.
- **Modular Architecture:** Clean backend code structure optimized for future web interface integration.
